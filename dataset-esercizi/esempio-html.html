Cos'è l'HTML

L'acronimo HTML sta per Hyper Text Markup Language ed è il principale linguaggio di pubblicazione di pagine Web.

Si tratta di un linguaggio di markup ('di marcatura'), che permette di indicare come disporre gli elementi all'interno di una pagina.

Queste indicazioni vengono date attraverso degli appositi marcatori, detti tag ('etichette'), che hanno la caratteristica di essere inclusi tra parentesi angolari.
